#==============================================================#
#                           CONFIG                             #
#==============================================================#
# TYPE           : The type of the channel                     #
# STATUS         : The status of the channel                   #
# ALIASES:       : The aliases of the channel                  #
# DISPLAY-COLOUR : The display colour of chat                  #
# FORMAT         : The format for chat, used to replace the    #
#                  default format                              #
# PASSWORD       : The password of the channel                 #
# RANGE          : The range of the channel                    #
# WHITELIST      : Whether whitelist is required               #
# DATA           : The data about the channel                  #
# BLACKLIST      : The blacklist                               #
# WHITELIST      : The whitelist                               #
# PARTICIPANTS   : The participants of the channel             #
#==============================================================#
#                            TYPES                             #
#==============================================================#
# standard - STANDARD CHANNELS BUILT IN TITANCHAT              #
#==============================================================#
#                          STATUSES                            #
#==============================================================#
# default - DEFAULT STATUS, PLAYERS JOIN CHANNELS WITH THIS    #
#           STATUS IF NOT IN ANY CHANNELS ON LOGIN             #
# none    - NO STATUS                                          #
# staff   - STAFF STATUS, ONLY STAFF CAN JOIN                  #
#==============================================================#
#                           RANGES                             #
#==============================================================#
# channel - CHANNEL RANGE, SEND CHAT TO PLAYERS IN THE CHANNEL #
#           AND TO THOSE WHO ARE FOLLOWING THE CHANNEL         #
# global  - GLOBAL RANGE, SEND CHAT TO ALL PLAYERS ON THE      #
#           SERVER                                             #
# local   - LOCAL RANGE, SEND CHAT TO ALL PLAYERS WITHIN A     #
#           BLOCK RADIUS OF THE PLAYER                         #
# world   - WORLD RANGE, SEND CHAT TO ALL PLAYERS IN THE SAME  #
#           WORLD                                              #
#==============================================================#
#                         COLOUR CODES                         #
#==============================================================#
# &0 (black), &1 (dark blue), &2 (green), &3 (dark aqua),      #
# &4 (red), &5 (purple), &6 (gold), &7 (silver),               #
# &8 (grey), &9 (blue), &a (light green), &b (aqua),           #
# &c (light red), &d (light purple), &e (yellow), &f (white),  #
# &k (magical), &l (bold), &m (strike through), &n (underline),#
# &o (italic), &r (reset)                                      #
#==============================================================#
